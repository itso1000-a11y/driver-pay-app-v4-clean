Driver Pay App v3

Included:
- Same working app logic from v2
- Save & Next navigation fix
- Backup / Restore in Settings
- Clear Current week button in the week selector
- start_driver_pay_app.bat for easier launch

How to run:
1. Open this folder.
2. Double-click start_driver_pay_app.bat.
3. If the browser does not open automatically, copy the localhost link from the black window.

Important:
- Keep using the same browser/address so local saved data stays visible.
- Use Backup in Settings before serious testing.
